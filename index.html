Pet Project

  • Online Sexism Detection Using Transformer [NLP]
    Using of advanced AI models, specifically transformer-based techniques like BERT and DistilBERT, to detect online sexism effectively and swiftly. Our deep learning approach aims to identify and mitigate sexist remarks on social media platforms, ensuring a safer and more respectful online environment.

  • Speech Emotion Recognition Using HMMs [Advance AI]
    We used Ryerson University's RAVDESS database to enhance speech emotion recognition. By analyzing key audio features with Hidden Markov Models (HMMs), they improved the nuances of emotional detection in speech. This work not only boosts the capabilities of virtual assistants like Siri and Alexa but also sets new standards for voice-assisted technology.

  • Comparative Analysis of Federated and Non federated CNN for Image Classification [ML]
    We explored the performance differences between federated and non-federated CNN models on the CIFAR-10 dataset, demonstrating that federated learning enhances privacy while achieving higher accuracy (59.62%) compared to traditional CNN approaches (56%). This work highlights the balance between model performance and data security in image classification tasks.

  • Study on Machine Learning Approach Using Fingerprint Recognition System [Advance Pattern Recognition]
    We focus on enhancing fingerprint recognition systems through neural networks. This work involves the use of ridge terminations and bifurcations as minutiae for fingerprint identification, emphasizing the impactful role of neural networks in improving recognition rates, especially when traditional methods are computationally intensive.

  • Image Manipulation and Real-Time Face Detection using OpenCV [CV]
    In the OpenCV project, basic image manipulation techniques such as adding text, drawing shapes, and applying transformations (rotation, scaling, translation) are explored. These foundational skills enable understanding of complex computer vision tasks, preparing for advanced projects like real-time face detection using OpenCV.
