Pet Projects

  • ECG-Based Outcome Prediction for Cardiac Arrest Patients Using 1D-CNN and Wavelet Transform [Biomedical Deep Learning]
    Developed a machine learning model to predict good or poor outcomes post-cardiac arrest using ECG data from the I-CARE database (PhysioNet). Preprocessed one-hour ECGs from 99 patients by resampling to 45 Hz, applying median and Butterworth filtering, and extracting time-frequency features via Continuous Wavelet Transform (Morlet). Segmented data into 30-second windows with 5-second overlaps, yielding 22,566 training and 5,642 validation samples. Trained a custom 1D-CNN with balanced precision, recall, and F1-scores.

  • Bengali Personality Traits Classification through LLM [NLP]
    I built a system to classify personality traits using Bengali text data. After preprocessing the dataset—handling missing values, tokenizing the text, and extracting features from audio—I fine-tuned GPT-2 (124M) and Bengali_BERT for better context understanding in text classification tasks. I then optimized the models to enhance their performance.

  • Defaulter & Loan Status Prediction Using Machine Learning Algorithms [ML]
    I built a machine learning-based system to predict loan approval status using applicant demographic and financial features. After preprocessing the dataset—handling missing values, encoding categorical variables, and scaling—I trained several models like Logistic Regression, SVC, Decision Tree, Random Forest, and Gradient Boosting, and evaluated them using K-Fold Cross-Validation, then hypertuned the models using Randomized Search to optimize their performance. The Random Forest classifier, and SVC gave the best results, so I saved it and integrated it into a Tkinter-based GUI for real-time predictions. This complete pipeline offers a practical tool for automating loan decisions in the banking domain.

  • Training Proximal Policy Optimization to LunarLander-v3 via Stable-Baselines3 [RL]
    Developed a reinforcement learning model utilizing the Proximal Policy Optimization (PPO) algorithm to train an agent in the LunarLander-v3 environment. The project encompassed configuring the environment, implementing the PPO algorithm using Stable-Baselines3, and evaluating the agent's performance. The trained model and associated files are hosted on Hugging Face's model hub, facilitating easy access and reproducibility.

  • Speech Emotion Recognition Using HMMs [Advance AI]
    We used Ryerson University's RAVDESS database to enhance speech emotion recognition. By analyzing key audio features with Hidden Markov Models (HMMs), they improved the nuances of emotional detection in speech. This work not only boosts the capabilities of virtual assistants like Siri and Alexa but also sets new standards for voice-assisted technology.

  • Comparative Analysis of Federated and Non federated CNN for Image Classification [ML]
    We explored the performance differences between federated and non-federated CNN models on the CIFAR-10 dataset, demonstrating that federated learning enhances privacy while achieving higher accuracy (59.62%) compared to traditional CNN approaches (56%). This work highlights the balance between model performance and data security in image classification tasks.

  • Online Sexism Detection Using Transformer [NLP]
    Using of advanced AI models, specifically transformer-based techniques like BERT and DistilBERT, to detect online sexism effectively and swiftly. Our deep learning approach aims to identify and mitigate sexist remarks on social media platforms, ensuring a safer and more respectful online environment.

  • Study on Machine Learning Approach Using Fingerprint Recognition System [Advance Pattern Recognition]
    We focus on enhancing fingerprint recognition systems through neural networks. This work involves the use of ridge terminations and bifurcations as minutiae for fingerprint identification, emphasizing the impactful role of neural networks in improving recognition rates, especially when traditional methods are computationally intensive.

  • Image Manipulation and Real-Time Face Detection using OpenCV [Computer Vision]
    In the OpenCV project, basic image manipulation techniques such as adding text, drawing shapes, and applying transformations (rotation, scaling, translation) are explored. These foundational skills enable understanding of complex computer vision tasks, preparing for advanced projects like real-time face detection using OpenCV.
